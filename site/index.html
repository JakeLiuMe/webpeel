How It Works

Smart escalation

Don't waste 2 seconds rendering when 118ms will do.
WebPeel auto-detects blocked requests and retries with the right tool.

1

HTTP Fetch

Fast HTTP fetch with smart headers and content extraction. Handles 80% of pages without a browser.

~2s

Features

Built for AI workflows

Smart Escalation

Tries HTTP first. Auto-escalates to headless browser only when needed. Fast by default.

Search Included

DuckDuckGo search built-in. Search the web and fetch results in one MCP call.

Clean Output

Strips nav, footer, ads, banners. Returns only the content your LLM actually needs.

Token Counting

Estimates token count so you can manage LLM context windows before sending.

Link Extraction

All links extracted as absolute URLs. Perfect for crawling and research workflows.

TypeScript-First

Full type definitions. Autocomplete and type safety out of the box.
